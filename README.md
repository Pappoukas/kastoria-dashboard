# 🏛️ Kastoria Tourism Visual Analytics Dashboard

> Διαδραστική εφαρμογή ανάλυσης τουριστικής εικόνας της Καστοριάς μέσω χρωματικής ανάλυσης φωτογραφιών και στατιστικής επεξεργασίας κριτικών TripAdvisor.

---

## 📋 Περιγραφή

Η εφαρμογή συνδυάζει δύο πηγές δεδομένων: τα **χρωματικά δεδομένα** από 1.604 φωτογραφίες επισκεπτών (εξαγμένα με k-means clustering σε χώρο Lab) και τις **κριτικές TripAdvisor** από 2.578 αξιολογήσεις για 18 αξιοθέατα της Καστοριάς. Οι δύο πηγές συνδέονται μέσω του μοναδικού αναγνωριστικού κάθε κριτικής, επιτρέποντας συσχετίσεις μεταξύ της οπτικής ταυτότητας των φωτογραφιών και της εμπειρίας των επισκεπτών.

Η ανάλυση καλύπτει: δημοφιλία αξιοθέατων, βαθμολογίες και τάσεις ικανοποίησης, εποχικότητα επισκεψιμότητας, γλωσσική και γεωγραφική προέλευση επισκεπτών, τύπο ταξιδιού, αλληλεπίδραση χρηστών, ανάλυση κειμένου με word clouds και sentiment, καθώς και πλήρη χρωματική ανάλυση (παλέτες, κορεσμός, φωτεινότητα, χρωματικότητα LCH, χώρος Lab) ανά αξιοθέατο.

## 🗂️ Αρχεία που απαιτούνται

| Αρχείο | Περιγραφή |
|--------|-----------|
| `color_summary_batch.xlsx` | Αποτελέσματα χρωματικής ανάλυσης (Summary, Clusters, Statistics) |

## ⚙️ Εγκατάσταση & Εκκίνηση

### Τοπικά (Python 3.9+)

```bash
git clone https://github.com/USERNAME/kastoria-dashboard.git
cd kastoria-dashboard
pip install -r requirements.txt
streamlit run app.py
```

Η εφαρμογή ανοίγει στο `http://localhost:8501`

### Streamlit Cloud (δωρεάν, χωρίς εγκατάσταση)

1. Ανέβασε το repo στο GitHub
2. Πήγαινε στο [share.streamlit.io](https://share.streamlit.io) → **New app**
3. Επέλεξε repo και `app.py` → **Deploy**

## 🔗 Σύνδεση δεδομένων

```
Excel[ID]  ←────────────────→  Reviews[id]
           (review ID join)

Reviews / Excel [placeInfo/name]  ←→  Attractions [placeInfo/name]
```

Κάθε φωτογραφία φέρει το `ID` της κριτικής από την οποία προέρχεται, επιτρέποντας άμεση σύνδεση χρωματικών δεδομένων με rating, γλώσσα, τύπο ταξιδιού κ.ά.
